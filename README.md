# 项目简介

## 核心功能
- 前后端代码及SQL文件
- 万字文档
- 参考导入调试视频
- 讲解视频
- 开发工具安装包

## 适用场景
- 企业员工考试管理
- 试卷发布与批改
- 成绩记录与查询
- 个人信息维护
- 留言与资讯交流

## 技术特点
- B/S架构，易于部署和维护
- MVC模式，代码结构清晰
- Java、SpringBoot、Vue技术栈，兼容性强
- 支持IDEA、JDK1.8、Mysql、Maven、Navcat等开发环境
